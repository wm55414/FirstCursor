# Windows Desktop App

An Angular application that simulates a Windows OS desktop with clickable folders and pictures.

## Features

- Windows-like desktop interface with blue gradient background
- Clickable folder and picture icons
- Single and double-click interactions
- Visual feedback on selection

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to `http://localhost:4200`

## Project Structure

- `src/app/desktop/` - Desktop component with Windows-like interface
- `src/app/app.component.ts` - Main app component
- `src/styles.css` - Global styles

## Development

- Run `npm start` for development server
- Run `npm run build` to build for production
